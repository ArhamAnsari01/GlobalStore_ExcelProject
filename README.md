# GlobalStore_ExcelProject

# Overview
This project analyzes a retail dataset containing over 50,000 transactions to understand sales performance, product profitability, and regional trends.

# Tools Used
Microsoft Excel

# Project Workflow
1. Cleaned raw CSV dataset
2. Created calculated columns (Profit Margin, Order Year, Order Month)
3. Built KPI metrics such as Total Sales and Profit Margin
4. Performed pivot table analysis
5. Developed an interactive dashboard using slicers

# Key Insights
1. Central region generated the highest sales and profit 
2. Copiers is the most profitable product category 
3. 2014 recorded the highest yearly sales 
4. December is the peak revenue month
